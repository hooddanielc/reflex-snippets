RE-flex Snippet
===============

## Dependencies

- [clang](https://clang.llvm.org/) - compiler
- [RE-flex](https://github.com/Genivia/RE-flex) - fast and flexible lexical analyzer generator
- [graphviz](https://www.graphviz.org/) - graph visualization software

## Generate Lexer

```
./gen.sh
```

## Build Executable

```
./build.sh
```

## Run

```
./run.sh
```
